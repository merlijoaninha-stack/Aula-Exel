# Aula-Exel
Frut-pague

Uma empresa de fruticultura e turismo  informações sobre espécies frutiferas, flora no geral e ecossistemas para publico leigo, expecialmente crianças, focando tanto na venda de produtos naturais sem consevantes ou corantes bem como em atvidades gratuitas e eventos voltados para a educação ambiental infantil.

Planilhas:

Controle de estoque dos produtos;

Épocas de flora e colheita;

Controle do publico.
